# Django test

## Install

Create admin user

```
python3 manage.py createsuperuser
```


## Run

```
python3 manage.py runserver
```
(or regular python)

Then visit:
```
http://127.0.0.1:8000/
```

## Admin

http://127.0.0.1:8000/admin
